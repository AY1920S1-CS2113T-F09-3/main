Directly modifies the value of a property of an item or item type.

edit:       edit stock <Stock Code> <Property> <New Value>
            edit stocktype <Stock Type> <New Value>
            *edit person <Matric No.> <Property> <New Value>

<Stock Code>        The model number of the existing stock
<Property>          The attribute of the stock. This includes "stocktype"/"stockcode"/"quantity"/"description".
<Stock Type>        The category name of the stock
<Quantity>          The number of stock in the inventory
<Description>       The description of the stock
<New Value>         The value of the property mentioned will be replaced by this new value

E.g. edit stock R500 quantity 100
E.g. edit stocktype resistant resistor
*E.g. edit person A012345 name Alex
